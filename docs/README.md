#### Opening the Document

#### Finalizing the Document (One-Time Setup)

**1. Generate Table of Contents**
   - Click on the "[Table of Contents - Auto-generated in Word]" placeholder
   - Go to `References` tab > `Table of Contents`
   - Select `Automatic Table 1` or `Automatic Table 2`
   - The TOC will populate with all headings and page numbers

**2. Add Page Numbers**
   - Go to `Insert` tab > `Page Number`
   - Select `Bottom of Page` > `Plain Number 2` (centered)
   - Go to page 2 (after cover page)
   - Check `Different First Page` to exclude cover from numbering

**3. Add Header**
   - Go to `Insert` tab > `Header` > `Blank`
   - Type: "VibeLink Ghana - Technical Documentation"
   - Format: Calibri 10pt, right-aligned
   - Check `Different First Page` to exclude cover

**4. Create Diagrams**
   The document includes placeholders for diagrams:
   - System Architecture Diagram (Technical Architecture section)
   - Git Workflow Diagram (Development Workflow section)

   To create these:
   - Go to `Insert` tab > `SmartArt` or `Shapes`
   - Create the diagram following the bullet points provided
   - Or use external tools (draw.io, Lucidchart) and insert as images

**5. Add Logo**
   - Replace "[Logo Placeholder]" on cover page
   - Go to `Insert` tab > `Pictures`
   - Select VibeLink logo file
   - Resize and center

**6. Review and Customize**
   - Review all content for accuracy
   - Update any project-specific information
   - Adjust formatting as needed
   - Save your changes
